Tuple :

   Tuple就像是串列（List），不過串列是可變動（Mutable）物件。
   
   而Tuple是不可變動（Immutable）物件。
   
   如果要排列必須先用list 在用Tuple。
   
     t1 = (3, 6, 1, 2, 7)
     l1 = list(t1)
     l1.sort()
     t1 = tuple(l1)
     t1
     ==>(1, 2, 3, 6, 7)
