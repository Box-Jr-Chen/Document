# xgboost

  - 官網 https://xgboost.readthedocs.io/en/latest/

  - xgboost(陳天琦) 提出，而後由幾十位博士維護升級。
  
  - gbdt 梯度提升樹的升級，透過殘差學習，改進成果。
   (一般情況下效果優於gbdt)
  
  - XGBoost是經過優化的分佈式梯度提升庫，旨在高效，靈活且可移植。它在Gradient Boosting框架下實現了機器學習算法。
  
  - 安裝`pip install xgboost`
